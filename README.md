# Algorithm Lab 🧬

Welcome to my Algorithm Lab repository! This collection contains Jupyter Notebooks documenting various algorithm implementations, complexity analyses, and lab exercises completed during my coursework.

## 🚀 Repository Overview

This lab focuses on the fundamental concepts of algorithm design, asymptotic analysis, and practical implementation using Python.

## 🛠️ Tools & Technologies

* **Language:** Python
* **Environment:** [Google Colab](https://colab.research.google.com/) / Jupyter Notebook
* **Library Focus:** Time complexity analysis, Matplotlib for growth visualization.

## 📈 Key Concepts Covered

* **Asymptotic Analysis:** Understanding $O(n)$, $\Omega(n)$, and $\Theta(n)$.
* **Recursion:** Solving recurrence relations using Master Theorem and Substitution.
* **Sorting & Searching:** Quick Sort, Merge Sort, and Binary Search.
* **Optimization:** Dynamic Programming and Greedy approaches.
